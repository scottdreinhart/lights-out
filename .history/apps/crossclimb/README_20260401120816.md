# Crossclimb

A graph search puzzle game where players navigate through interconnected nodes to find the optimal path from start to finish while collecting checkpoints along the way.

## Game Overview

Crossclimb is a pathfinding puzzle that challenges players to find the shortest path through a graph while collecting all required checkpoints. The game features multiple difficulty levels with increasingly complex graph structures and AI-powered hints and solutions.

## Features

- **Graph-based Gameplay**: Navigate through nodes connected by weighted edges
- **Checkpoint System**: Collect all checkpoints to complete the puzzle
- **Multiple Algorithms**: AI uses BFS, DFS, and A* search for optimal solutions
- **Interactive Canvas**: Visual graph rendering with clickable nodes
- **Hint System**: Get suggestions for the next optimal move
- **Complete Solution**: View the entire optimal path
- **Responsive Design**: Works on desktop and mobile devices
- **Accessibility**: Keyboard navigation and screen reader support

## Difficulty Levels

- **Easy**: Small graphs with 8-12 nodes, simple connectivity
- **Medium**: Medium graphs with 12-16 nodes, moderate complexity
- **Hard**: Large graphs with 16-20 nodes, complex connectivity patterns

## How to Play

1. **Start**: Begin at the green start node
2. **Navigate**: Click adjacent nodes to build your path
3. **Collect Checkpoints**: Visit all yellow checkpoint nodes
4. **Finish**: Reach the red end node with all checkpoints collected
5. **Optimize**: Try to find the shortest possible path

## Controls

- **Mouse/Touch**: Click nodes to move
- **Hint Button**: Shows the next optimal move
- **Solve Button**: Reveals the complete optimal solution
- **Reset Button**: Start a new game

## Technical Implementation

### Architecture

- **Domain Layer**: Pure game logic, graph algorithms, AI solvers
- **App Layer**: React hooks for state management and game orchestration
- **UI Layer**: Canvas-based rendering and user interface components

### Algorithms

- **BFS (Breadth-First Search)**: Finds shortest path in unweighted graphs
- **DFS (Depth-First Search)**: Explores graph depth-first
- **A* Search**: Optimal pathfinding with heuristic guidance

### Technologies

- **React**: UI framework with hooks
- **TypeScript**: Type-safe development
- **Canvas API**: Interactive graph visualization
- **CSS Modules**: Scoped component styling
- **Vitest**: Unit testing framework

## Game Logic

The core game mechanics are implemented in the domain layer:

- **Graph Generation**: Creates random connected graphs with varying complexity
- **Path Validation**: Ensures moves are valid and paths are connected
- **Checkpoint System**: Requires collection of all checkpoints for completion
- **AI Solvers**: Multiple algorithms for finding optimal paths
- **Difficulty Scaling**: Adjusts graph size and connectivity based on level
