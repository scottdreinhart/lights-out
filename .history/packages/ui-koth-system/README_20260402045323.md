# @games/ui-koth-system

A shared UI system for King-of-the-Hill (KotH) competitive game rankings and leaderboards.

## Overview

`@games/ui-koth-system` provides reusable React components and hooks for displaying KOTH-style leaderboards where a single player is the "king" and must defend their position against challengers.

## Features

- **Ranking Screen** — Full-screen display of current rankings with player scores and positions
- **Leaderboard Component** — Reusable leaderboard entry list with ranking positions
- **Individual Entries** — Styled ranking entries with player info and scores
- **Responsive Design** — Works across mobile (xs/sm), tablet (md), desktop (lg), and ultrawide (xxl) viewports
- **Accessibility** — WCAG 2.1 AA compliant with keyboard navigation and semantic HTML
- **Theme Support** — Integrates with shared theme system via CSS variables
- **Type-Safe** — Full TypeScript support

## Usage

### Using the Enhanced Data Hook

The `useKothData` hook provides localStorage-backed persistence and automatic ranking:

```tsx
import { useKothData } from '@games/ui-koth-system'

export const GameResultsScreen = () => {
  const { entries, addEntry, getPlayerRank } = useKothData({
    gameName: 'sudoku',
    maxEntries: 100,
  })

  const handleGameEnd = (score, duration) => {
    addEntry({
      username: 'PlayerName',
      score,
      duration,
      difficulty: 'medium',
      wins: 1,
      timestamp: Date.now(),
      rank: 0,
    })
  }

  return (
    <div>
      {entries.slice(0, 10).map((entry) => (
        <KothEntryRow key={entry.id} entry={entry} />
      ))}
    </div>
  )
}
```

### Display Podium with Top 3

```tsx
import { KothPodium, useKothData } from '@games/ui-koth-system'

export const PodiumDisplay = () => {
  const { entries } = useKothData({ gameName: 'chess' })
  const [first, second, third] = entries.slice(0, 3)

  return <KothPodium first={first} second={second} third={third} />
}
```

## Components

### KothPodium

Visual podium display for top 3 entries with achievements and medal badges.

**Props:**

- `first?: KothEntry` — First place entry (gold medal 👑)
- `second?: KothEntry` — Second place entry (silver medal 🥈)
- `third?: KothEntry` — Third place entry (bronze medal 🥉)
- `accentColor?: string` — Custom accent color for styling (default: `#0087be`)

**Example:**

```tsx
import { KothPodium } from '@games/ui-koth-system'

const TopThree = ({ topEntries }) => {
  const [first, second, third] = topEntries
  return (
    <KothPodium
      first={first}
      second={second}
      third={third}
      accentColor="#ff5722"
    />
  )
}
```

### KothEntryRow

Individual ranking row with metadata (duration, difficulty, wins, etc.).

**Props:**

- `entry: KothEntry` — The ranking entry to display
- `isCurrentPlayer?: boolean` — Highlight current player (default: `false`)
- `accentColor?: string` — Custom accent color (default: `#0087be`)

**Features:**

- Rank display with `#` prefix
- Player name with "You" badge for current player
- Score with locale formatting
- Duration formatting (e.g., "2h 15m")
- Difficulty metadata
- Wins counter (if applicable)
- Relative time display ("just now", "5m ago", "2h ago", etc.)

**Example:**

```tsx
import { KothEntryRow } from '@games/ui-koth-system'

const LeaderboardList = ({ entries, currentUsername }) => {
  return (
    <div>
      {entries.map((entry) => (
        <KothEntryRow
          key={entry.id}
          entry={entry}
          isCurrentPlayer={entry.username === currentUsername}
          accentColor="#09a3d5"
        />
      ))}
    </div>
  )
}
```

### KothRankingScreen

Full-screen ranking display component.

**Props:**

- `playerScore: number` — Current player's score
- `playerRank: number` — Current player's rank (1 = king)
- `onNewGame: () => void` — New game callback
- `onHome: () => void` — Home/menu callback
- `entries?: KothEntry[]` — Optional leaderboard entries to display

**Example:**

```tsx
<KothRankingScreen
  playerScore={2500}
  playerRank={3}
  onNewGame={handleNewGame}
  onHome={handleHome}
  entries={leaderboardData}
/>
```

### KothLeaderboard

Reusable leaderboard list component.

**Props:**

- `entries: KothEntry[]` — Array of ranking entries
- `currentPlayerId?: string` — ID of current player (for highlighting)
- `onSelectEntry?: (entry: KothEntry) => void` — Entry selection callback

**Example:**

```tsx
<KothLeaderboard entries={topPlayers} currentPlayerId={userId} onSelectEntry={showPlayerProfile} />
```

### KothRankingEntry

Individual ranking entry component.

**Props:**

- `rank: number` — Player's rank (1 = king)
- `playerName: string` — Player's display name
- `playerScore: number` — Player's score
- `isCurrentPlayer?: boolean` — Highlight current player
- `isKing?: boolean` — Show king indicator
- `avatar?: string` — Optional player avatar URL

**Example:**

```tsx
<KothRankingEntry
  rank={1}
  playerName="SuperSkills"
  playerScore={5000}
  isKing={true}
  isCurrentPlayer={false}
/>
```

## Hooks

### useKothLeaderboard

Hook for managing KOTH leaderboard state and operations.

**Returns:**

```tsx
{
  entries: KothEntry[]
  currentRank: number
  currentScore: number
  kingName: string
  isKing: boolean
  loadEntries: (data: KothEntry[]) => void
  updateScore: (score: number) => void
}
```

**Example:**

```tsx
import { useKothLeaderboard } from '@games/ui-koth-system'

export const MyComponent = () => {
  const { entries, currentRank, isKing, updateScore } = useKothLeaderboard()

  return (
    <div>
      <p>Your Rank: {currentRank}</p>
      <p>{isKing ? '👑 You are the KING!' : 'Challenge the king!'}</p>
      <KothLeaderboard entries={entries} />
    </div>
  )
}
```

## Responsive Behavior

Components are fully responsive across all device tiers:

- **Mobile** (xs/sm: <600px) — Single column, touch-optimized, full-width buttons
- **Tablet** (md: 600–899px) — Balanced layout, side-by-side buttons where helpful
- **Desktop** (lg: 900–1199px) — Full-featured layout with spacing
- **Widescreen** (xl: 1200–1799px) — Extra spacing, more generous padding
- **Ultrawide** (xxl: 1800px+) — Maximum refinement, centered max-width container

All components adapt automatically using `window.matchMedia()` (via shared hooks) or CSS media queries.

## Performance

- Leaderboard lists are virtualized for large datasets (1000+ entries)
- CSS animations use `transform` and `opacity` for 60fps performance
- Lazy loading for entry avatars
